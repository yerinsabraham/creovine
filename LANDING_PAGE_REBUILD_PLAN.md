## Mobile Responsiveness Requirements

### Breakpoints
- Desktop: 1024px+
- Tablet: 768px - 1023px
- Mobile: 320px - 767px

### Mobile-Specific Adjustments
1. **Typography**
   - Hero H1: 32px ‚Üí 24px
   - Body text: 16px ‚Üí 14px
   - Buttons: min 44px height (thumb-friendly)

2. **Spacing**
   - Reduce padding/margins by 50%
   - Increase line-height for readability

3. **Navigation**
   - Hamburger menu
   - Full-screen overlay menu
   - Close button

4. **Tables**
   - Stack rows vertically
   - Card-style layout instead of table

5. **Code Snippets**
   - Horizontal scroll
   - Smaller font size
   - Syntax highlighting maintained

6. **Images**
   - Responsive sizing
   - Lazy loading
   - WebP format

7. **Forms (Auth Modal)**
   - Full-screen on mobile
   - Large input fields
   - Auto-focus and keyboard handling
